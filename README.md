# solar-cargo-be
### 1. Create .env 
```
    POSTGRES_DB=solarCargoDb
    POSTGRES_USER=solarCargoDbUser
    POSTGRES_PASSWORD=solarCargoSuperSecretDbPassword
```
### 2. Build and start containers

```
    docker compose up --build
```
